# My neovim config
Spared no expense

### Install
```
curl -s https://raw.githubusercontent.com/whleucka/neovim-config/main/install.sh -H "Cache-Control: no-cache, no-store" | bash -s
```

### Uninstall
```
unlink $HOME/.config/nvim
```
